🔬 연구 개요
숏폼 플랫폼(틱톡·릴스·쇼츠)의 급격한 성장과 함께, 
사용자의 지루함 역치(Boredom Threshold) 와 일상 집중력 간의 관계를 행동 데이터로 직접 측정합니다.

기존 연구들이 자기보고 설문에만 의존했던 한계를 극복하고, 
실제 행동 측정(스킵 타이밍, 반응속도, 인지 테스트) 을 통해 인과 관계를 분석합니다.
